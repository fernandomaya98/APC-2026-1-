O Little Man Computer é um site que permite simular uma CPU funcionando e usa a linguagem Assembly, linguagem de baixo nível que se assemelha mais a linguagem de máquina. O programa que eu programei faz uma soma e uma subtração sequencial, o programa recebe como entrada três números: X,Y,Z. A máquina vai acessar o X na memória, somar o número Y com o número X e armazenar o novo valor de X na memória. Após isso, a máquina vai novamente acessar o número X na memória, subtrair o número Z com o número X e armazenar o novo valor de X na memória. Por fim, a máquina acessa o valor final de X na memória e apresenta ele como saída.
